# MarketSim
A tool for viewing market data and experimenting with investment strategies.

## Features
- View real-time and historical stock market data
- Create and manage multiple accounts and portfolios
- Execute trades with virtual cash
- Search for stocks to customize an account's watchlist

## Installation
```bash
git clone https://github.com/Teafed/marketsim.git
```
## Configuration
MarketSim requires API keys for [Finnhub](https://finnhub.io/) and [Massive](https://massive.com/). Launch with keys as environment variables, or keep them in a .env file:
```
FINNHUB_API_KEY=your_key
POLYGON_API_KEY=your_key
```
